Canvace.Audio.SourceNode Class

Defined in: src\Audio.js:97
Module: Canvace

This class represents a sound resource that the browser is capable of playing.

If the browser supports WebAudio, this class wraps around an AudioSourceNode; if it doesn't, this class wraps around an HTMLAudioElement.

You cannot instantiate this class directly: you can obtain a new instance by using the Audio.load, Loader.getSound or Loader.playSound method.
