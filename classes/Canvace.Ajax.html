_delete

  • [load]
  • [type]
)
Canvace.Ajax.Request static

Defined in src/Ajax.js:441

Issues a DELETE HTTP request and returns a corresponding new Request object.

Parameters:

  • url Mixed

    This first parameter is either a string representing the URL of the requested resource, or a dictionary of options to pass to the constructor of Request.

    See its documentation for more details about the allowed options.

  • [data] Object optional

    Optional custom parameters to send to the server. Ignored if the first parameter is not a string.

    See the Canvace.Ajax.Request constructor for more information.

  • [load] Function optional

    An optional callback function that gets called as soon as the server response arrives and the request completes successfully. Ignored if the first parameter is not a string.

    See the Canvace.Ajax.Request constructor for more information.

  • [type] String optional

    Indicates how the browser should interpret the resource contents. Ignored if the first parameter is not a string. Defaults to an empty string.

Returns:

Canvace.Ajax.Request:

The instantiated request object.

Example:

Canvace.Ajax._delete('/threads', {
    id: 1234
}, function (response) {
    if (response.success) {
        alert('Thread deleted');
    } else {
        alert(response.message);
    }
});

Canvace.Ajax.Request static

Defined in src/Ajax.js:481

Retrieves a resource by using a GET HTTP request and interprets its contents as JSON.

Returns a new Request object.

Parameters:

  • url String

    The URL of the requested JSON resource.

  • [onLoad] Function optional

    The callback function to invoke when the loading is complete. See the onLoad method for details.

  • [onError] Function optional

    The callback function to invoke when the loading aborts with an error. See the onError method for details.

Returns:

Canvace.Ajax.Request:

The instantiated request object.

Example:

Canvace.Ajax.getJSON('/data/stage1.json', function (stageData) {
    // ...
}, function (statusCode, statusText) {
    alert(statusText);
});
