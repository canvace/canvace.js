Canvace.List.Accessor Class

Defined in: src/List.js:35
Module: Canvace

Provides access to list elements.

This class cannot be instantiated directly, instances are returned by methods of the outer class.

element

() Any

Defined in src/List.js:56

Returns the accessed element.

Returns:

Any:

The accessed element.

Example:

var list = new Canvace.List();
var accessor = list.addTail('Hello, world!');
alert(accessor.element()); // alerts "Hello, world!"

next

() Canvace.List.Accessor

Defined in src/List.js:104

Returns an Accessor for accessing the next element of the list, or null if this Accessor represents the last element.

An exception is thrown if the element represented by this accessor has been removed by this or any other accessor.

Returns:

Canvace.List.Accessor:

An Accessor for accessing the next element, or null if this Accessor represents the last element.

Example:

var list = new Canvace.List();
list.addTail(1);
list.addTail(2);
list.addTail(3);
// the following loop alerts, in order: 1, 2, 3
for (var a = list.getHead(); a; a = a.next()) {
    alert(a.element());
}

previous

() Canvace.List.Accessor

Defined in src/List.js:70

Returns an Accessor for accessing the previous element of the list, or null if this Accessor represents the first element.

An exception is thrown if the element represented by this accessor has been removed by this or any other accessor.

Returns:

Canvace.List.Accessor:

An Accessor for accessing the previous element, or null if this Accessor represents the first element.

Example:

var list = new Canvace.List();
list.addTail(1);
list.addTail(2);
list.addTail(3);
// the following reverse loop alerts, in order: 3, 2, 1
for (var a = list.getTail(); a; a = a.previous()) {
    alert(a.element());
}

remove

() Boolean

Defined in src/List.js:137

Removes the element from the list.

Returns:

Boolean:

true if the element was successfully removed, false otherwise. false is returned if this method is called more than once, thus trying to remove the same element multiple times: only the first call returns true.

Example:

var list = new Canvace.List();
list.addTail(1);
list.addTail(2);
list.addTail(3);
for (var a = list.getHead(); a; a = a.next()) {
    if (a.element() === 2) {
        a.remove();
    }
}
// the element 2 has been removed
