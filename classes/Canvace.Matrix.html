Canvace.Matrix Class

Defined in: src/Matrix.js:21
Module: Canvace

Represents a general purpose three-dimensional matrix use to store generic data indexed by I, J and K coordinates. The data is stored efficiently since the Matrix class greatly reduces memory footprint.

This class is used by the engine to manage tile maps and other internal three-dimensional data.

Enumerates all the elements in the matrix and calls the specified callback function for each one.

The enumeration can be interrupted by the callback function returning false, in which case the forEach method returns true.

Enumerates the layers in the matrix. A layer is a set of elements in the matrix that have the same k coordinate.

The enumeration can be interrupted by the callback function returning false, in which case the forEachLayer method returns true.

Parameters:

  • callback Function

    A user-defined function that is invoked for each layer in the matrix.

    • k Number

      The current layer number, or k coordinate, passed to the callback function at each iteration.

  • [scope] Object optional

    An optional object that is used as this when invoking the callback function.

Returns:

Boolean:

true if the enumeration was interrupted by the callback function return false; false if all the layers were enumerated.
